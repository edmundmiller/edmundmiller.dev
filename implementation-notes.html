Dither Kit Implementation Notes

Updated: 2026-07-11

Design decisions

  • Decision: Add one compact writing-cadence area chart near the post list.
    Why: It visualizes existing site content without creating a demo page or turning the text-first home page into a dashboard.

Deviations from spec

  • Deviation: Stabilized dither-kit series registration and added strict TypeScript guards inside the generated engine.
    Why: The registry source assumes React Compiler memoization and looser indexed-access settings; without the callback fix, the hydrated Astro island entered an infinite update loop.

Tradeoffs considered

  • Options: Homepage writing cadence, about-page career timeline, or standalone demo.
    Chosen: Homepage writing cadence.
    Reason: Post dates provide honest local data and the chart directly supports the adjacent content.
